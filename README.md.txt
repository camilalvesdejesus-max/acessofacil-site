# AcessoFácil - E-commerce Acessível

Projeto desenvolvido para a segunda entrega do Projeto Integrador.

## Objetivo

Desenvolver um site de e-commerce acessível voltado para pessoas com deficiência, idosos e usuários com baixa familiaridade com tecnologia, promovendo inclusão digital, acessibilidade, autonomia e facilidade de navegação.

## Páginas do site

- Página inicial
- Página de produtos
- Página do carrinho

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Exibição de produtos acessíveis
- Carrinho de compras funcional
- Cálculo automático do total
- Finalização via WhatsApp
- Navegação simplificada
- Layout responsivo

## Recursos de acessibilidade

- Letras ampliadas
- Alto contraste
- Botões ampliados
- Estrutura semântica HTML
- Navegação por teclado
- Texto alternativo em imagens
- Links de acessibilidade
- Foco visível em elementos interativos
- Compatibilidade com leitores de tela
- Layout responsivo

## Melhorias realizadas

Durante o desenvolvimento foram aplicadas melhorias baseadas na avaliação do ASES (Avaliador e Simulador de Acessibilidade em Sítios), incluindo:

- Criação de atalhos de acessibilidade
- Ajustes semânticos no HTML
- Remoção de eventos inline “onclick”
- Implementação de “addEventListener”
- Melhorias de foco visual
- Ajustes de navegação e usabilidade

## Responsividade

O site foi desenvolvido para funcionar em computadores, tablets e smartphones.
